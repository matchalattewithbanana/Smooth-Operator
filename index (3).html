Autoplay Engine
Spotify-style BaRT algorithm — works for any track, any language, any genre, even completely unknown content. Enable with -autoplay.
Metadata-driven — never depends on keyword lists or genre databases to function. Those signals improve results when detected, but even with zero matches it still works using only the raw data from Lavalink: artist name, title, duration, source, and session history.
BaRT Strategy
Exploit (70%) — Familiar
Same artist, artist hits, seed title + artist, genre-aware artist queries, most-played session artist. Always works from raw metadata.
Explore (30%) — Discovery
Every 3rd call: genre-aware searches, "fans also like", title similarity, session diversity query, source-aware angle. Falls back gracefully when nothing is detected.
Session Dedup
Hard URI dedup — same song never repeated in a session (up to 50 tracks). Resets automatically when full, like Spotify restarting radio.
Random Top-N
Never picks the first result blindly. Selects randomly from top 3–5 fresh results so even repeated queries give variety.
Pure Metadata Signals (no keywords needed)
Top Artist
Counts artist frequency across the last 6 tracks. Most-played artist shapes exploit queries. Pure count — no keywords.
Duration Heuristic
When genre detection fails: short tracks (<2.5min) → pop/rap angle · long tracks (>6min) → ambient/classical angle. Uses raw length from Lavalink.
Session Diversity
1 artist → dig deeper (discography) · 2–3 artists → crossover mix · 4+ artists → compilation mode. Built from author fields only.
Source-Aware
Spotify source → chart angle · SoundCloud → underground · Deezer/Apple → official · YouTube → recommended. Uses sourceName from Lavalink.
Language + Genre Detection (optional enhancement)
Detected from track titles and artist names when keywords match. Improves explore queries by adding language context. English is the default — never needs explicit detection.
